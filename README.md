# AI4SoilHealth Toolbox – Quarto starter site

This is a starter Quarto website for turning the AI4SoilHealth toolbox brochure into a collaborative documentation site.

## What is included

- website scaffold (`_quarto.yml`)
- core pages for the toolbox narrative
- initial tool catalogue pages
- starter styling in `styles.css`
- starter images and crops prepared from the material already shared in chat

## Suggested next steps

1. Create a repository under the AI4SoilHealth GitHub organization.
2. Copy these files into the repo.
3. Install Quarto locally and preview the site.
4. Replace temporary images with better tool-specific images where available.
5. Send individual tool pages to responsible partners for correction.
6. Connect deployment to a subdomain such as `toolbox.ai4soilhealth.eu`.

## Suggested review order

- first: agree on site structure and wording of the toolbox definition
- second: review the tool pages with tool owners
- third: refine graphics and contact page
